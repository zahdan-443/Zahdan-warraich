# Public Assets Directory

Put your custom app photos and logos inside this `public` folder!

### Required File Names:
1. **`logo.png`** (App Logo - Recommended size: 512 x 512 px)
2. **`splash.png`** (Splash Screen Background - Recommended size: 1080 x 1920 px)

Any file placed directly in this folder can be accessed in web browsers starting with `/` (for example: `/logo.png`).
